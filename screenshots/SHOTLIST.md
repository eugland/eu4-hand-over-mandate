# Screenshot shot list

Drop screenshots in this folder. Suggested filenames below — gallery shows them in upload order, so number them.

In-game capture: F11 (EU4 screenshot, saves to `Documents/Paradox Interactive/Europa Universalis IV/screenshots/`) or F12 (Steam). Move files here afterwards.

## Must-have (sells the mod)

1. `01_decision.png` — Decisions tab as Ming with the **Hand Over the Mandate** decision visible (hover it so the tooltip with conditions shows).
2. `02_candidate_menu.png` — The abdication event with all 8 **"Offer the Mandate to …"** options listed. Console: `event ham_events.1` as Ming.
3. `03_offer.png` — The **"An Offer from the Son of Heaven"** event from the recipient's side (Accept / Decline buttons). Console: fire the offer, then `tag KOR` (or whoever you offered to) to see their popup.
4. `04_new_emperor.png` — After acceptance: new holder's government view showing **Celestial Empire / Emperor of China**, ideally with the China map behind. Proof it works.

## Nice-to-have

5. `05_tributaries.png` — **"The Mandate Has Passed"** event with the release-tributaries / keep-tributaries choice.
6. `06_opinion.png` — Diplomacy screen opinion breakdown showing **"Grateful for the Mandate +50"** or **"Passed Over for the Mandate −25"**.
7. `07_refusal.png` — **"The Offer Is Refused"** event (pick-another / give-up options).
8. `08_take_mandate_cb.png` — Declare-war screen as a non-East-Asian, non-pagan/eastern neighbor (e.g. Russia bordering the Emperor) showing the **Take Mandate of Heaven** CB available — the opened-CB feature.

## Notes

- PNG or JPG, under ~8 MB each; 1920×1080 fine.
- This folder ships inside the mod upload (harmless, a few MB). Delete it before uploading if you want a lean package.
